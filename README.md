# spurfaden
spurfaden - app

## Website hosting

This repository now includes a minimal static website in `/website` and a GitHub Actions workflow that deploys it to GitHub Pages.

### How to use

1. Put your website files in `/website` (replace `index.html` as needed).
2. In GitHub, enable Pages for this repository and set **Build and deployment** to **GitHub Actions**.
3. Push to `main` to trigger deployment.
